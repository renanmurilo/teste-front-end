# Teste Front-end

## Faça o clone do repositório

## Para instalar o projeto
```
npm install
```

### Para rodar o projeto
```
npm run serve
```

### Para compilar para produção
```
npm run build
```

### Para fazer teste
```
npm run test
```

### Para fazer correções
```
npm run lint
```
